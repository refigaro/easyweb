Marca

El logotipo de Easyweb combina la mascota (monstruito azul) con el wordmark en gris oscuro. La tilde del símbolo "ý" es el elemento cyan que referencia la paleta primaria de la marca.

Motion

Animar solo transform y opacity. Nunca transition-all. Pasa el cursor sobre las tarjetas para ver cada easing en acción.

Spring

cubic-bezier(0.34, 1.56, 0.64, 1)
Smooth
--ease-smooth
cubic-bezier(0.4, 0, 0.2, 1)
Ease Out
--ease-out
cubic-bezier(0, 0, 0.2, 1)

Token Reference

Referencia completa de todos los tokens CSS. Copia el bloque de variables y agrégalo al :root de cualquier archivo para heredar el sistema.

/* === EASYWEB DESIGN TOKENS v1.0 === */
:root {
  /* Brand Colors */
  --brand-primary:        #21D2F2;
  --brand-primary-dark:   #0891b2;
  --brand-primary-light:  #cffafe;
  --brand-cta:            #02C000;
  --brand-cta-hover:      #16a34a;

  /* Text */
  --text-primary:         #333333;
  --text-secondary:       #4b5563;
  --text-muted:           #9ca3af;
  --text-on-dark:         #ffffff;

  /* Surfaces */
  --surface-base:         #ffffff;
  --surface-subtle:       #f9fafb;
  --surface-muted:        #f3f4f6;
  --surface-dark:         #333333;

  /* Typography */
  --font-display:         'Work Sans', sans-serif;
  --font-body:            'Work Sans', sans-serif;
  --font-mono:            'JetBrains Mono', monospace;

  /* Type Scale */
  --text-3xl:   6rem;
  --text-2xl:   4.5rem;
  --text-xl:    4rem;
  --text-lg:    3.25rem;
  --text-md:    2.25rem;
  --text-sm:    2rem;
  --text-lead:  2rem;
  --text-summary: 1.5rem;
  --text-body:  1.125rem;

  /* Spacing */
  --space-1: 0.25rem;  --space-2: 0.5rem;   --space-3: 0.75rem;
  --space-4: 1rem;     --space-6: 1.5rem;   --space-8: 2rem;
  --space-10: 2.5rem;  --space-12: 3rem;   --space-16: 4rem;
  --space-24: 6rem;    --space-32: 8rem;

  /* Border Radius */
  --radius-sm: 0.25rem;  --radius-md: 0.5rem;   --radius-lg: 0.75rem;
  --radius-xl: 1rem;    --radius-2xl: 1.5rem; --radius-3xl: 2rem;
  --radius-full: 9999px;

  /* Shadows */
  --shadow-xs:  0 1px 2px 0 rgba(0,0,0,.05);
  --shadow-sm:  0 2px 8px -1px rgba(0,0,0,.08);
  --shadow-md:  0 4px 20px -2px rgba(0,0,0,.10);
  --shadow-lg:  0 8px 32px -4px rgba(0,0,0,.14);
  --shadow-glow-cyan:  0 0 24px rgba(33,210,242,.35);
  --shadow-glow-green: 0 0 24px rgba(2,192,0,.30);

  /* Motion */
  --ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);
  --ease-smooth: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-out:    cubic-bezier(0, 0, 0.2, 1);
  --duration-fast:   150ms;
  --duration-base:   250ms;
  --duration-slow:   400ms;
  --duration-slower: 600ms;
}
